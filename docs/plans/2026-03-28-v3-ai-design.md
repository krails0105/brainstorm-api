# v3.0 AI 기능 설계 문서

## 개요

v3.0 마일스톤 (4/13 ~ 4/26). Python AI 마이크로서비스로 LLM 기반 키워드 도출/확장, 아이디어 확장 기능 구현.

## 아키텍처

```
FE (React) → Spring Boot BE → Python AI Service (LangChain + FastAPI)
                                    ↓
                              LLM API (Claude/GPT/etc, 교체 가능)
```

- BE ↔ AI Service 통신: REST API (HTTP)
- LLM 모델: 환경변수로 선택 가능하게 설계 (vendor lock-in 방지)

## AI 기능 3가지

### 1. 키워드 기반 확장 키워드 생성

기존 키워드를 넣으면 유사/확장 키워드를 제안.

**Python API:** `POST /ai/keywords/expand`

Request:
```json
{
  "keywords": ["AI", "번역"],
  "topic": "스타트업 아이디어"
}
```

Response:
```json
{
  "keywords": ["자동화", "NLP", "실시간 번역", "다국어 지원"]
}
```

### 2. 채팅 기반 자동 키워드 도출

대화 내역에서 주제 관련 키워드를 자동 추출. 최대 12개.

**Python API:** `POST /ai/keywords/extract`

Request:
```json
{
  "chatHistory": "A: 요즘 번역 서비스 많이 쓰는데...\nB: AI로 실시간 번역하면 좋겠다...",
  "topic": "스타트업 아이디어"
}
```

Response:
```json
{
  "keywords": ["번역", "AI", "실시간", "서비스"]
}
```

### 3. 기존 아이디어 확장

키워드 + 대화 내역 + 기존 아이디어를 종합해서 새 아이디어/솔루션 도출.

**Python API:** `POST /ai/ideas/expand`

Request:
```json
{
  "keywords": ["AI", "번역", "실시간"],
  "chatHistory": "...",
  "ideas": ["AI 기반 번역 서비스"],
  "topic": "스타트업 아이디어"
}
```

Response:
```json
{
  "ideas": ["AI 실시간 자막 번역 서비스", "다국어 회의 자동 통역"],
  "solutions": ["WebSocket 기반 실시간 번역 스트리밍", "음성→텍스트→번역 파이프라인"]
}
```

## Spring Boot BE 변경

### 새로 추가할 클래스

| 클래스 | 역할 |
|--------|------|
| AiController | FE 요청 받아서 AI 서비스 호출 중개 |
| AiService | RestTemplate/WebClient로 Python AI 서비스 HTTP 호출 |
| AiKeywordRequest / AiKeywordResponse | 키워드 확장/도출 DTO |
| AiIdeaRequest / AiIdeaResponse | 아이디어 확장 DTO |

### BE API

| 메서드 | 경로 | 설명 |
|--------|------|------|
| POST | `/api/rooms/{roomId}/ai/keywords/expand` | 키워드 확장 (기존 키워드 → AI → 새 키워드) |
| POST | `/api/rooms/{roomId}/ai/keywords/extract` | 채팅→키워드 도출 (대화 내역 → AI → 키워드) |
| POST | `/api/rooms/{roomId}/ai/ideas/expand` | 아이디어 확장 (키워드+대화+아이디어 → AI → 새 아이디어) |

### 흐름 예시 (채팅 → 키워드 도출)

```
1. FE: POST /api/rooms/{roomId}/ai/keywords/extract
2. BE AiController: ChatMessage DB에서 대화 내역 조회
3. BE AiService: Python AI 서비스에 POST /ai/keywords/extract
4. Python: LangChain으로 LLM 호출 → 키워드 추출
5. BE: 응답 받아서 Keyword 엔티티로 저장
6. FE: 생성된 키워드 목록 반환
```

## Python AI Service 구조

```
brainstorm-ai/
├── main.py              # FastAPI 서버
├── routes/
│   ├── keyword.py       # 키워드 확장/도출 API
│   └── idea.py          # 아이디어 확장 API
├── services/
│   ├── keyword_service.py
│   └── idea_service.py
├── prompts/             # LLM 프롬프트 템플릿
├── config.py            # 모델 설정 (환경변수)
├── requirements.txt
└── .env                 # LLM API 키
```

### Tech Stack

- FastAPI — Python REST API 프레임워크
- LangChain — LLM 호출 + 프롬프트 관리
- LLM: Claude/GPT 등 교체 가능 (환경변수)

### 환경변수

| 변수 | 설명 |
|------|------|
| LLM_PROVIDER | "claude" / "openai" / "gemini" |
| LLM_API_KEY | LLM API 키 |
| LLM_MODEL | 모델명 (예: claude-sonnet-4-5, gpt-4o) |
